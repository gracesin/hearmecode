Getting started with Python and running your first program


Windows/Linux

Mac

  • Download and install Python 2.7.6 from http://www.python.org/download/releases/2.7.6/

  • Search for IDLE in your programs and open IDLE

  • In IDLE, go to File > New (or CTRL + N) to make a new program

  • Type the following command into the screen on the left:

    print “Watch out, world!”

  • Save your program.

    • File > Save (or CTRL + S) brings up the Save dialog box

    • Select the location where you want to save the file

    • Give the file a name that ends with .py, e.g. lesson1.py

  • Press F5 to run the program (or Run menu > Run module)

  • Look at results on the right side of your screen

  • Python comes bundled with Macs – no downloading and installing needed!

  • Download and install Sublime Text 3 from http://www.sublimetext.com/3

  • Open Sublime

  • Type the following command:

    print “Watch out, world!”

  • Save your program.

    • CMD + S brings up the Save dialog box

    • Select the location where you want to save the file

    • Give the file a name that ends with .py, e.g. lesson1.py

  • Press CMD + B to run your programs

  • Look at results at the bottom of the screen
